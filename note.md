# 测试速率
# TCP
- 服务器端: `iperf3 -s -p 5001`
- 客户端: `iperf3 -c 192.168.1.100 -p 5001`

# UDP
- 服务器端: `iperf3 -s -p 5001`
- 客户端: `iperf3 -c 192.168.1.100 -p 5001 -u`

# 测试指标
- 带宽: 稳定时的网络带宽, 对于TCP还需要保证数据传输的可靠性
 - 单个连接的带宽: 每秒传输的字节数
 - 多个连接的带宽
- RTT时延: 往返时延
- 并发量: 同时进行的连接数量
- 网卡的吞吐量: 不考虑协议处理
- PPS: 每秒处理的数据包数量

- 不同的请求大小, 随着并发量的增加, 总的吞吐量和时延的变化情况.
